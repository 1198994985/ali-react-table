示例

fusion

表格拓展功能依赖组件库，本页面中的示例使用了 fusion 组件库 作为组件库。示例中的 <assets.FusionStyles /> 用于动态加载 fusion 的 CSS 样式。

暗色模式

因为 fusion 对暗色模式的兼容性不佳，推荐使用亮色模式浏览当前页面。

行展开#

Live Editor点击展开或收拢
function 行展开() {
  const { dataSource5, columns5 } = assets.biz
  const pipeline = useTablePipeline({ components: fusion })
    .input({
      // dataSource 结构：
      //  [ { id: '1', title: '阿里巴巴网络技术有限公司', children: [...] },
      //    { id: '2', title: '蚂蚁金服有限公司', children: [...] },
      //    { id: '3', title: '其他', children: [...] }, ]
      dataSource: dataSource5,
      columns: columns5,
    })
    .primaryKey('id')
    // 受控用法：rowGrouping({ openKeys, onChangeOpenKeys })
    .use(features.rowGrouping({ defaultOpenKeys: ['1', '2'] }))
  return (
    <>
      <assets.FusionStyles />
      <BaseTable {...pipeline.getProps()} />
    </>
  )
}

树形可选择表格#

Live Editor点击展开或收拢
function 树形可选择表格() {
  const dataSource = assets.biz.dataSource4
  const columns = assets.biz.columns4
  const pipeline = useTablePipeline({ components: fusion })
    .input({ dataSource, columns })
    .primaryKey('id')
    .use(features.treeMode())
    .use(
      features.treeSelect({
        tree: dataSource,
        rootKey: 'root',
        defaultValue: ['1', '3'],
        checkboxColumn: { lock: true },
        highlightRowWhenSelected: true,
      }),
    )
  return (
    <>
      <assets.FusionStyles />
      <BaseTable {...pipeline.getProps()} />
    </>
  )
}
